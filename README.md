# Heise Ladebordstein Analyse

Dieses Repository enthält eine lokale Auswertung von drei Heise-Forenthreads zum Thema Ladebordsteine.

Enthalten sind:

- das ausführbare Skript `heise_forum_curbcharger_analyse.ps1`
- Vollberichte und geglättete Endfassungen für drei Threads
- JSON-Auswertungen pro Thread
- eine Gesamtstatistik über alle drei Threads
- eine Vergleichsdatei `Köln_vs_TankE_vs_Serie`

Die grossen HTML-Caches und Laufzeitdateien sind bewusst nicht versioniert.
